Portfolio Project

Book Catalog Platform

Web と Android の両方から利用できる書籍管理アプリです。公開検索、認証付きのお気に入り管理、 管理者向けの蔵書追加・削除までを一つのサーバレス構成にまとめています。

Web / Android Amazon Cognito API Gateway + Lambda DynamoDB

Capabilities

  • タイトル・著者名ごとの検索モード切り替え
  • 認証済みユーザー向けのお気に入り登録 / 削除
  • 管理者グループの追加・削除操作
  • Terraform ベースのインフラ管理
